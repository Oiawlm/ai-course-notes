## 中文

# AI 课程笔记

直播课程的原始文字稿 + 高质量结构化笔记。

使用 [transcript-to-detailed-notes](https://github.com/Oiawlm/transcript-to-detailed-notes) skill 处理生成。

### 结构

```
文字稿/    ← 原始飞书导出
笔记/      ← skill 处理后的结构化笔记
```

### 命名格式

`时间-标题-作者-平台`

- 时间: 8 位数字，如 `20260616`
- 标题: 直播主题
- 作者: 分享人
- 平台: 小鹅通 / 视频号 等
